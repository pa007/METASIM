\subsection{Detailed Description}
The base class for every simulation object. An entity has an internal status, an interface for modyfing the status, and can contain one or more events. An entity can be referred also by its name (a string of characters) using the static method {\itshape find}. A specific entity class should redefine the {\itshape find} function for doing type checking.
