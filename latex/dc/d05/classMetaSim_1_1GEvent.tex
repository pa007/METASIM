\subsection{Detailed Description}
\subsubsection*{template$<$class X$>$\\*
class Meta\+Sim\+::\+G\+Event$<$ X $>$}

This template class is used for simplifying the writing of a system model. Most of the times, the \hyperlink{classMetaSim_1_1GEvent_a50bac7d037bcce48edb2cf2a866ef021}{doit()} method of an event must only call a handler method of some class. Normally, you must write a new class derived from event and override the virtual \hyperlink{classMetaSim_1_1GEvent_a50bac7d037bcce48edb2cf2a866ef021}{doit()} method. However, it is a lot of repetitive work! And when there is a lot of repetitive work to do, template is the solution!

The \hyperlink{classMetaSim_1_1GEvent}{G\+Event} class takes a class derived from \hyperlink{classMetaSim_1_1Entity}{Entity} as a template argument. Then, template function register\+\_\+handler() must be called passing the pointer to the handler before using this event.
